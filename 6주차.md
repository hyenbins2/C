# 6주차
- 6주차 실습

```
#include <stdio.h>

int main() {
  int n, sum = 0, i = 1;
  printf("정수를 입력하시오:");
  scanf("%d", &n);

  while (i <= n) {
    sum += i;
    i++;
  }

  printf("1부터 %d까지의 합은 %d입니다.", n, sum);
  return 0;
}
```

```
#include <stdio.h>

int main(void) {
  int n, sum = 0, i = 1;
  printf("정수를 입력하시오: ");
  scanf("%d", &n);

  while (i <= n) {
    if (i % 2 == 0) {
      sum += i;
    }
    i++;
  }
  printf("1부터 %d까지의 짝수합은 %d입니다.", n, sum);
}
```
