# 8주차 복습 
 - 이중 for문 (* 찍기)  https://www.acmicpc.net/problem/2439
 

1. 이중 for문 code 
```.C
#include <stdio.h>

int main(){
	int n;

	scanf("%d", &n);

	for(int i = 0; i < n; i++){
		for(int j = 0; j < n; j++) {
            if(j < n - i - 1) printf(" ");
			else printf("*");
		}
		printf("\n");
	}
	return 0;
}
```
